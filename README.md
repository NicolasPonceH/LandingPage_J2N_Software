# 📖 Contextualización

**J2N Software** es una agencia de desarrollo enfocada en construir soluciones tecnológicas del mundo real.

Esta Landing Page fue construida para presentar de forma visual, profesional e interactiva todos los servicios y proyectos del equipo, incluyendo:

- 💻 Desarrollo de Aplicaciones Web a medida
- 🤖 Integración de Inteligencia Artificial
- 🌐 Sistemas IoT (Internet of Things)
- 📱 Desarrollo de Plataformas Móviles
- 🚀 Portafolio interactivo de proyectos destacados

La página utiliza un diseño futurista con toques "holográficos" y animaciones dinámicas que demuestran la modernidad, capacidad técnica e innovación de los desarrolladores detrás de J2N Software.

---

# ✨ Características

- 🎬 Animaciones fluidas utilizando Framer Motion.
- ⚛️ Partículas dinámicas (BorderParticles) y botones interactivos (RadioactiveButton).
- 📹 Video holográfico 3D en la sección principal (Hero).
- 🧩 Diseño completamente modular basado en componentes de React.
- 📱 Diseño 100% Responsive para todos los dispositivos.
- 🌙 Interfaz en modo oscuro ("Dark Theme") con gradientes y efectos de neón.
- 💼 Sección de portafolio y servicios estructurados.
- 🛠 Visualización interactiva del Stack Tecnológico.
- 📬 Sección de contacto integrada.

# 🛠 Tecnologías Utilizadas

# ⚙️ Instalación

## 1️⃣ Clonar el repositorio

```bash
git clone https://github.com/NicolasPonceH/LandingPage_J2N_Software.git
```

## 2️⃣ Ingresar al proyecto

```bash
cd LandingPage_J2N_Software
```

## 3️⃣ Instalar dependencias

```bash
npm install
```

## 4️⃣ Ejecutar el entorno de desarrollo

```bash
npm run dev
```

## 5️⃣ Abrir en el navegador

```text
http://localhost:5173
```
*(El puerto por defecto de Vite)*
